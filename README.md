# Customer Segmentation System

This project is a web-based customer segmentation and recommendation system developed using HTML, CSS, and JavaScript.

## Features
- Customer segmentation using predefined rules
- Displays customer category based on Customer ID
- Provides marketing recommendations
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open index.html in a web browser
3. Enter a valid Customer ID to see the result
